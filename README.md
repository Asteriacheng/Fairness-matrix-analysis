# Fairness-matrix-analysis
This project investigates credit risk prediction and fairness evaluation in data-driven lending systems using logistic regression models. A random sample of 5,000 observations from the Home Credit Default Risk dataset was used, split into training and test sets. Two logistic regression models were constructed: a baseline model including core financial and stability indicators, and an extended model incorporating demographic and contextual risk features. Model performance was assessed using ROC curves and AUC metrics to evaluate ranking ability under class imbalance. In addition, fairness diagnostics were conducted across demographic groups by analysing approval rate disparities, error rate differences, and calibration properties. The results highlight the trade-off between predictive accuracy and fairness considerations in credit scoring models.
